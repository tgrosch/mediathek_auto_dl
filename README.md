mediathek_auto_dl
=================

Mediathek Auto Downloader


## Requirements
* python2
* pip2
* virtualenv

## Installation (Linux/Mac)

    ./scripts/install.sh

## Usage (Linux/Mac)
    . activate
    ./manage.py runserver
* Open your favorite browser and enter the following address:

    http://localhost:8000/

## Installation (Windows)

    scripts/install.bat

## Usage (Windows)
    activate.bat
    python manage.py runserver
* Open your favorite browser and enter the following address:

    http://localhost:8000/
